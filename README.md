# Kigali Travel Guide

**Live Site:** https://travel-kigali.netlify.app/

A simple website I built to showcase interesting places in Kigali. This was my HTML & CSS assignment where I focused on making a responsive site that works well on phones and computers.

## What's Inside

The site has 5 pages covering different spots around Kigali:

- **Home** - Quick intro and navigation
- **Museum** - Info about the Kigali Genocide Memorial
- **Park** - Details on Nyarutarama Park
- **Restaurant** - Menu and hours for Heaven Restaurant  
- **Market** - Guide to Kimironko Market

## How I Built It

Just HTML and CSS - no frameworks or libraries. I wanted to keep it simple and focus on:
- Making it look good on mobile (most people browse on phones anyway)
- Using clean colors instead of too many gradients
- Adding real travel photos from Unsplash
- Making sure everything loads fast

## Running It Locally

1. Clone or download this repo
2. Open `index.html` in your browser
3. That's it - no build process needed

## Things I Learned

This project helped me practice:
- CSS Grid and Flexbox for layouts
- Media queries for responsive design
- Working with forms, tables, and embedded content
- Organizing CSS with variables
- Structuring a multi-page site

## Deployment

Hosted on Netlify because it's free and super easy - just drag and drop your files.
