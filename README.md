# Kigali Guide Website

## About This Project
I created this website to show different places you can visit and eat in Kigali. I wanted to make something useful for tourists and people who want to explore the city.

## Pages in the Website
- **Home Page** - Welcome page with navigation to all sections
- **Museum Page** - About Kigali Genocide Memorial with visiting hours
- **Park Page** - Information about Nyarutarama Park and activities
- **Restaurant Page** - Heaven Restaurant menu and prices
- **Market Page** - Kimironko Market sections and what you can buy

## What I Learned
While building this website, I practiced:
- Structuring pages with HTML tags
- Making tables for organizing information
- Adding images and styling them with CSS
- Embedding videos and audio
- Creating navigation menus with hover effects
- Using flexbox to arrange elements
- Making forms for user contact
- Linking pages together
