# Kigali Guide Website

## About This Project
A modern, responsive website showcasing the best places to visit, eat, and explore in Kigali, Rwanda's beautiful capital city. This website is designed to be useful for tourists and anyone wanting to discover what Kigali has to offer.

## Features
✨ **Fully Responsive Design** - Works perfectly on mobile, tablet, and desktop devices
🎨 **Modern & Beautiful UI** - Clean design with attractive color schemes for each page
🖼️ **High-Quality Images** - Professional images from Unsplash
🎥 **Multimedia Content** - Embedded videos and audio guides
📱 **Mobile-First Approach** - Optimized for all screen sizes
🚀 **Deployment Ready** - Clean code ready for hosting on any platform

## Pages in the Website
- **Home Page** - Welcome page with hero section, featured places, video, and contact form
- **Museum Page** - Kigali Genocide Memorial with visiting hours, gallery, and audio guide
- **Park Page** - Nyarutarama Park with activities, rules, pricing, and video tour
- **Restaurant Page** - Heaven Restaurant with menu, prices, hours, and ambiance
- **Market Page** - Kimironko Market with sections, products, and shopping tips

## Technologies Used
- **HTML5** - Semantic markup with proper meta tags
- **CSS3** - Modern styling with:
  - CSS Variables for consistent theming
  - Flexbox and Grid for layouts
  - Media queries for responsive design
  - Smooth transitions and hover effects
  - Gradient backgrounds
  - Box shadows and modern effects

## Design Features
- **Consistent Navigation** - Sticky navigation bar across all pages
- **Color Themes** - Each page has its own unique color palette:
  - Home: Green & Blue (Rwanda colors)
  - Museum: Brown & Gold (Memorial theme)
  - Park: Green (Nature theme)
  - Restaurant: Orange (Food theme)
  - Market: Purple (Vibrant market theme)
- **Responsive Tables** - All tables scroll horizontally on mobile
- **Card Layouts** - Modern card-based designs for content
- **Interactive Elements** - Hover effects and smooth transitions

## Responsive Breakpoints
- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px
- **Small Mobile**: Below 480px

## How to Deploy

### Option 1: GitHub Pages
1. Push your code to a GitHub repository
2. Go to Settings > Pages
3. Select your branch and save
4. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify
1. Create account at netlify.com
2. Drag and drop your project folder
3. Your site will be live instantly with a custom URL

### Option 3: Vercel
1. Create account at vercel.com
2. Import your GitHub repository
3. Deploy with one click

### Option 4: Traditional Hosting
Upload all files to your web hosting via FTP:
- index.html
- All .html files
- All .css files
- All image files (.jpg)
