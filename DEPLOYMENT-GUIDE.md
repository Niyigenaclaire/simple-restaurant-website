# 🚀 Deployment Guide for Kigali Guide Website

## Quick Start - Choose Your Platform

### 1️⃣ GitHub Pages (FREE & EASY)

**Steps:**
1. Create a GitHub account at https://github.com
2. Create a new repository (name it: kigali-guide)
3. Upload all your files to the repository
4. Go to Settings → Pages
5. Under "Source", select your main branch
6. Click Save
7. Your site will be live at: `https://yourusername.github.io/kigali-guide`

**Time:** 5-10 minutes  
**Cost:** FREE

---

### 2️⃣ Netlify (EASIEST - DRAG & DROP)

**Steps:**
1. Go to https://netlify.com
2. Sign up for free account
3. Click "Add new site" → "Deploy manually"
4. Drag your entire project folder into the upload area
5. Wait 30 seconds
6. Your site is live!

**Time:** 2-3 minutes  
**Cost:** FREE  
**Bonus:** You get a custom URL like `kigali-guide.netlify.app`

---

### 3️⃣ Vercel (FAST & PROFESSIONAL)

**Steps:**
1. Go to https://vercel.com
2. Sign up with GitHub
3. Click "New Project"
4. Import your repository
5. Click "Deploy"
6. Done!

**Time:** 3-5 minutes  
**Cost:** FREE  
**Bonus:** Automatic deployments when you update code

---

### 4️⃣ Traditional Web Hosting

**If you have web hosting (like Hostinger, Bluehost, etc.):**

1. Connect to your hosting via FTP (use FileZilla)
2. Upload all files to the `public_html` folder
3. Your site will be live at your domain

**Files to upload:**
- index.html
- museum.html
- park.html
- restaurant.html
- market.html
- home.css
- museum.css
- park.css
- restaurant.css
- market.css
- All .jpg image files
- README.md (optional)

## ✅ Pre-Deployment Checklist

Before deploying, make sure:
- [ ] All pages load correctly locally
- [ ] All links work (test navigation)
- [ ] Images display properly
- [ ] Videos and audio work
- [ ] Forms are functional
- [ ] Site is responsive (test on phone)
- [ ] No broken links
- [ ] README is updated

---

## 📱 Testing Your Deployed Site

After deployment, test:
1. **Desktop view** - Open in Chrome, Firefox, Safari
2. **Mobile view** - Test on your phone or use browser dev tools
3. **Tablet view** - Use browser responsive mode
4. **All pages** - Click through every page
5. **All links** - Make sure navigation works
6. **Media** - Check videos, audio, and maps load

## 🆘 Need Help?

**Common Issues:**
- **Images not showing:** Make sure image files are uploaded
- **CSS not loading:** Check file names match exactly (case-sensitive)
- **404 errors:** Ensure index.html is in the root folder
- **Mobile issues:** Clear browser cache and test again
